# 🤖 AQuil AI - Productivity Assistant  
  
AQuil AI adalah chatbot AI yang dirancang untuk membantu Anda meningkatkan produktivitas.  
  
## 🚀 Fitur Utama  
  
- ✅ Mengerjakan tugas dengan efisien  
- ✅ Berpikir kritis tentang masalah  
- ✅ Memberikan motivasi dan dukungan  
- ✅ Mengoreksi kesalahan dengan bijak  
- ✅ Real-time chat experience  
  
## 📋 Requirements  
  
- Python 3.8+  
- Flask  
- Flask-SocketIO  
- Transformers (Hugging Face)  
- PyTorch  
- NumPy  
  
## 🛠️ Installation
